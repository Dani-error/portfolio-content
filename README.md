# Dani B. Castro Portfolio Content

This repository contains the content of my personal portfolio [danibcastro.me](https://danibcastro.me), organized as Markdown files for articles and projects.

## Repository Structure

```
root/
├── articles/ # Markdown files for blog posts and articles
└── projects/ # Markdown files for portfolio projects
```


- **`articles/`**: Contains all the written content for my portfolio's blog or knowledge base.  
  Example: `articles/my-first-article.md`

- **`projects/`**: Contains Markdown files for each project showcased on my portfolio.  
  Example: `projects/awesome-project.md`

## How to Use

1. **View locally:** You can open any Markdown file with your preferred editor or Markdown viewer.  
2. **Integration:** These files are used in my portfolio website to dynamically generate the content for articles and projects.  
3. **Contribution:** If you'd like to contribute, please submit a pull request with your Markdown content in the appropriate folder.
